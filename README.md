# Next Best Read Recommendation Engine

A recommendation engine for avid readers who use Goodreads. This application helps you conquer your "want to read" list by providing intelligent, personalized, and timely book recommendations based on your own curated selection.

## Features

*   **Goodreads Integration:** Easily import your library via the standard Goodreads export CSV.
*   **Customizable Recommendations:** Set preferences for genre, book length, publication year, and more.
*   **Weighted Scoring:** Uses a smart algorithm to prioritize books based on your criteria, ratings, and recency.
*   **Export:** Generate a CSV to update your Goodreads shelves with a "recommended-for-you" tag.

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```

3.  **Usage:**
    *   Export your library from Goodreads (My Books -> Import and export -> Export Library).
    *   Upload the CSV file to the application.
    *   Set your preferences (mood, length, etc.).
    *   Get your top 5 recommendations!
